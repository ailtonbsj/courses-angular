# History of commands

```bash
# Start serve of hotreloader
npm start

# Install material design package
ng add @angular/material

# Generate Header component
ng g c components/template/header

# Generate Footer component
ng g c components/template/footer

# Generate Nav component
ng g c components/template/nav

# Generate Home component
ng g c views/home

# Generate ProductCrud component
ng g c views/product-crud

# Generate ProductCreate component
ng g c components/product/product-create

# Generate Red directive
ng g d directives/red

# Generate For directive
ng g d directives/for

# Generate Product service
ng g s components/product/product

# Generate ProductRead component
ng g c components/product/product-read

# Generate table with schematics
ng generate @angular/material:table components/product/product-read2
```

# Commands

```bash
# Dexie Todo list
ng g m dexie-todo
npm install dexie
ng g i dexie-todo/TodoList
ng g i dexie-todo/TodoItem
ng g c dexie-todo/Todo
ng g c dexie-todo/ItemList

# Quagga Barcode scanner
ng g m quagga-barcode
ng g c quagga-barcode/scanner
npm install --save @ericblade/quagga2

ng serve --host=0.0.0.0 --disable-host-check
ssh -R 80:localhost:4200 nokey@localhost.run
```
